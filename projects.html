5G Network KPI Analysis

Conducted an experimental study on the impact of weather conditions on 5G Key Performance Indicators (KPIs) such as latency, signal strength, and throughput. Developed a real-time data collection system using a Raspberry Pi with a 5G hat, integrating environmental data from WeatherAPI. Processed data using Python, Plotly, and Matplotlib, visualizing KPIs with interactive heatmaps to correlate temperature, humidity, and precipitation with network performance.

+

BlueLink Bluetooth & WiFi IoT Network

Designed and implemented a distributed IoT network using ESP32 microcontrollers connected via Bluetooth and WiFi. The system enables real-time control and automation of:

  • Smart lights with RGB control
  • Smart fans and thermostats
  • Air quality monitoring
  • Flood water detection
Integrated with a user-friendly Android app for seamless smart home automation.
Watch project overview video.

Extended Capacity Uninterrupted Power Supply

Developed a Deep Cycle Battery UPS system as a senior design project under the guidance of Dr. Jennifer L. Zirnheld. The project involved designing a cost-effective and efficient UPS solution, supporting a 600W load using a 100Ah battery. One model incorporated solar charging capabilities for extended operation.

Pong in VHDL (EE478)

Designed and implemented a hardware-accelerated version of Pong using VHDL on an FPGA board. This project showcases digital logic, finite state machines (FSMs), and VGA signal generation.

Tetris in C++ (EE379)

Developed a fully functional Tetris game in C++, featuring real-time game logic, collision detection, and smooth rendering. The project demonstrates efficient graphics programming and structured software design.

Pinnacle Hackathon 2021

Created Prism, an augmented reality project using Magic Leap for holographic manufacturing assistance. This project combined 3D printing, CAD design, and AR visualization, earning an exclusive invite to Pinnacle Hackathon.

UB Hackathon 2020 - 2022

Co-developed Red Alert, a fire prevention system using servos, circuits, and a backend server. The project won the $2,000 Grand Prize and the Best Hardware award at UB Hackathon 2020.

Printer Filament Recycling (Code Green)

Developed a plastic recycling system to convert waste bottles into 3D printer filament, winning the Best Hardware Hack and Hack with a Purpose awards.
